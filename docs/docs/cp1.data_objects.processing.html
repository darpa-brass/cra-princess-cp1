compute_value(x)¶

Computes value as a function of bandwidth. If the provided bandwidth is less than the total minimum bandwidth, which is the sum of the safety and voice bandwidth requirements, this TA provides 0 value. It is impossible to assign over 2000 Kbps to a given TA, so this equation caps out at that value.

Parameters:x (int) – The amount of bandwidth allocated to this TA.
