cp1.processing.algorithms.discretizer.accuracy_discretization module¶

accuracy_discretization.py

Provides a gurantee on the accuracy of the overall solution in relation to the optimal solution. Guaranteed accuracy is: 1 - accuracy i.e. accuracy = 0.001, therefore guaranteed accuracy is 99.999 disc_count = (1 - 0.001) * (100 / ta.min_value)

Author: Tameem Samawi (tsamawi@cra.com)
